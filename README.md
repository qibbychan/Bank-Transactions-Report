# Bank-Transactions-Report
This project focuses on building an end-to-end Power BI solution for bank transaction monitoring. The dashboard provides a high-level overview of transaction trends while incorporating advanced features to detect fraudulent patterns and manage transaction details efficiently.

**Key Features**
- Transaction Summaries: Displays total transactions and total transaction amounts (e.g., $771.17K).
  
- Fraud Identification: Categorizes and flags suspicious activities using "True/False" indicators.

- Global Mapping: Visualizes transaction distribution across various geographic locations using latitude and longitude data.

- Interactive Filters: Includes slicers for Transaction Type, Status, Device Used, and Network Slice ID for tailored data views.
